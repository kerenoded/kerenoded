## Open Source Projects

Most of my public projects focus on **practical tooling for testing and operating distributed systems in AWS environments**.

### [AWS Fargate Workload Runner](https://github.com/kerenoded/aws-fargate-workload-runner) [![Stars](https://img.shields.io/github/stars/kerenoded/aws-fargate-workload-runner?style=social&logo=github&v=2)](https://github.com/kerenoded/aws-fargate-workload-runner/stargazers)
